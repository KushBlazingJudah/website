WSJ

This is just a non-exhaustive list of my views. It is in no particular order. These focus on mostly technology and were written in the middle of the night a while ago. It may be incoherent at times.

  • Politics
    • Might as well get this one out of the way.
    • I hate talking about them with people publicly. It's kinda cringe. In private, yeah sure whatever that's cool. I know who I'm talking to. But there's no reason to sperg out on what you think is right or wrong.
    • I try to stay apolitical in most public conversations, because no matter what you'll always have someone pissed off at you and somebody praising you. This isn't to say that I have been political in some conversations, but I try not to be political when possible and try to steer away from political conversations when I'm taking place in them. If I don't just leave the conversation as a whole. Seeing someone say "I support X and Y!" in a public chat and then watching it get flooded with "HOLY BASED" and "kill yourself, I'll explain why and how" gets boring after a while.
  • Social media
    • For reference and more understanding, I am going to define social media here as having the defining features of the current popular social media applications. This includes things like scores, likes/shares, ads, and being generally privacy invasive and FOSS-hostile. However, it is not limited to just those features and I have special exclusions for some services. I consider Discord a social media despite not having any of these features except the last two, but IRC and Matrix not.
    • It is a cancer in our current society and I actively steer as far away as possible from it.
    • But like a cancer, sometimes it shows up at my door and I gotta deal with it unwillingly. Sometimes it's hard to remove.
    • Deleting most of my socials might've cost me a little bit on my friendships, but what I lose there I gain in sanity.
    • It's not good when it's the first thing you see when you wake up and the last thing before bed.
    • I am more than aware that I run a Discord server. I want to move away but won't for now, however my position as the Discord administrator might eventually be up for grabs.
    • I have not used a social media service that has not left a sour taste in my mouth. All of them suck.
  • Piracy
    • It ain't stealing if you still have whatever I took.
    • It is up to me on what to do with my copy of my files. I own them, don't I?
    • Copyright is kinda a meme anyway. Copyleft is where it's at.
  • DRM
    • Wake me up when you figure out a reason for DRM besides "proving ownership", because with DRM you never actually own what you buy.
  • Streaming services
    • What's the point if I end up paying the same amount as cable when I get everything I want?
    • I can't even watch the news unless I stack on the bundles!
    • The only use for it is "watch it when you want, provided your internet is good and we have it licensed". I'll pirate it, thank you. I've had much better viewing experiences watching Hackers.1996.1080p.WEB-DL.YIFY.mp4 than streaming it.
  • Software
    • Free software is based and redpilled. Proprietary software is cringe and bluepilled.
    • I currently run almost 100% free software on my computers. Notable exceptions are Discord, and the others are shitty web apps from Google. I have yet to pry myself away from the official YouTube website and onto Invidious. I do dual-boot Windows but the damn thing barely even runs on my 2016 (read: recent) laptop that came with Windows!
    • Write simple software. Write a program to do one thing, and do it well. Read up on the UNIX philosophy. (HTTP) I don't always follow these rules but I try to when I can.
    • I will write a page on my current software stack eventually.
  • Web browsers
    • I only use Firefox because it isn't a derivative of Chromium. There is nothing viable that doesn't use some derivative of Chromium's engine anymore.
    • Code should not be executed in the web browser. If your site doesn't work without JavaScript, make it work without it. I should not need JavaScript to read an article. If you're gonna write an app, suck it up and make it run natively instead of using a gimped Chromium build.
    • Speaking about that, Electron is cringe. I don't need to run six (6) web browsers on my computer, I need one. Don't write code for a gimped web browser. Write it natively. I don't care if it's harder.
